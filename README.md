# Login Page

A simple login page built using HTML and CSS.

## Files Included

- `index.html`: Contains the structural markup for the login form.
- `styles.css`: Contains the styling rules for the UI.

## How to Run

Since this is a static webpage, no server is required to view it. 

1. Simply open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge, etc.).
2. Alternatively, you can use a local server like VS Code's "Live Server" extension for a better development experience.

## Customization

You can easily modify the design by editing `styles.css` and adjust the form structure in `index.html` to suit your project's needs.
